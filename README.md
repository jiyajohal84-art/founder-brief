# Founder Brief Generator

An AI-powered investment research tool built for pre-seed VC workflows.

Paste in a founder name, company, and any context — get a structured investment brief in seconds, including market analysis, founder signals, questions to probe, and red/amber/green flags.

Built with the Claude API (claude-sonnet-4-6). No backend, runs entirely in the browser.

**Live demo:** https://idyllic-sherbet-4b9cc8.netlify.app

---

Built in a day to understand Playfair's sourcing workflow before applying for their AI engineering internship.
